# WebHost

This is the final work developed for the subject Advanced Topics in Computer Science (TAI). It consists of a system that manages a reseller of hosts.

___
### Technologies used:

- PHP
- Composer
- Laragon
- MySQL
- Laravel
- Bootstrap
- Heroku
