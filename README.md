# WebHost

Este é o trabalho final desenvolvido para a disciplina de Tópicos Avançados em Informática (TAI). Ele consiste em um sistema que gerencia uma revenda de hosts.

___
### Tecnologias utilizadas

- PHP
- Laragon
- Heidi SQL
- Laravel
- Bootstrap
- Heroku
